### Technologies used
-   HTML, CSS, SASS, React

### How to run the app
-   Clone the repo
-   Run `npm install` or `yarn`
-   Copy `.env.example` to `.env` and setup the appropriate information
-   Run `npm run dev` or `yarn dev`
